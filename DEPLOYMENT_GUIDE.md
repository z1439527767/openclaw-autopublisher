# TikTok应用部署指南

## 🏆 推荐方案：GitHub Pages

### 部署步骤
1. 创建GitHub仓库：`openclaw-autopublisher`
2. 上传`tiktok-deploy`文件夹内容到仓库
3. 开启GitHub Pages
   - Settings → Pages → Source: `main` branch

### 获得URL
```
https://[your-username].github.io/openclaw-autopublisher/
```

### 使用此URL
在TikTok开发者平台填写：
- **Web/Desktop URL**: `https://your-username.github.io/openclaw-autopublisher/`
- **隐私政策**: `https://your-username.github.io/openclaw-autopublisher/privacy-policy.html`
- **使用条款**: `https://your-username.github.io/openclaw-autopublisher/terms-of-service.html`

## 🌐 Vercel部署（需要账号）

### 步骤
1. 注册Vercel账号：https://vercel.com
2. 上传`tiktok-deploy`文件夹
3. 自动获得URL：`https://openclaw-autopublisher.vercel.app/`

## 🔧 手动Vercel部署

1. 下载tiktok-deploy文件夹
2. 压缩为zip文件
3. 访问：https://vercel.com/new
4. 拖拽zip文件上传
